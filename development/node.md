## Path

* see global module path
    ```
    npm bin -g
    ```
* add global module path to PATH
    ```
    export PATH=`npm bin -g`:$PATH
    ```
* add local module path to PATH
    ```
    export PATH=`npm bin`:$PATH
    ```
