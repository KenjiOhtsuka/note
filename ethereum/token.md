# Token

## ERC20

FungibleTokenをSolidityで開発する際に推奨されているトークンの仕様

例: EOS, TRON

## ERC721は、Non-FungibleTokenをSolidityで開発する際に推奨されているトークンの仕様

例: CryptoKitties
