# Spanish

## 冠詞

### 定冠詞

| | 男性 | 女性 |
|:--|:--|:--|
| 単数 | el | la |
| 複数 | los | las |

### 不定冠詞

| | 男性 | 女性 |
|:--|:--|:--|
| 単数 | un | una |
| 複数 | unos | unas |

## 人称代名詞

### 単数

* 1人称: y
* 2人称
    * 親しい仲: tú
    * 敬称: usted - 文法上は3人称として扱う
* 3人称
    * 男性: él
    * 女性: ella

### 複数

* 1人称
    * 一般: nosotros
    * 女性のみ: nosotras
* 2人称
    * 一般: vosotros
    * 女性のみ: cosotras
    * 敬称: ustedes - 文法上は3人称として扱う
* 3人称
    * 一般: ellos
    * 女性のみ: ellas

## 動詞

### ESTAR

...の状態である

|  | 単数 |  | 複数  |  |
|:--|:--|:--|:--|:--|
| 1人称 | y | estoy | nosotros / nosotras | estamos |
| 2人称 | tú | estás | vosotros / vosotras | estáis |
| 3人称 | usted; él / ella | está | ustedes; ellos / ellas | están |
