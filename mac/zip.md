# Zip

## Zip

`zip`: zip ファイルを作成するコマンド。


## ZipCloak

`zipcloak`: zip ファイル にパスワードをつけるためのコマンド。

## MacZip4Win

mac の zipcloak を使ってパスワードをつけたものが開けないと連絡を受けたので、
こちらを使ってパスワードを付与した。

### install

```
brew cask install maczip4win
```

### How to use

GUI Window に、 zip にしたいファイルをドラッグ&ドロップする。
