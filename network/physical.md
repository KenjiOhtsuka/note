# Physical Network

## LAN Cable

* LANケーブルにタグをつけてわかりやすくする。
* 表: 黒でここを表す文字を
* 裏: 赤で行き先を表す文字を

```
(Black)    (Red)
---------  ---------
| From  |  |  To   |
---------  ---------
   [ ]        [ ]
    |          |
    |          |
    |          |
    |          |
    |          |
    |          |
    |          |
    |          |
```

* 例
    * 黒: PC3, 複合機1 
    * 赤: L1-1/4
