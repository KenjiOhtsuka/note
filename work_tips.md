# Work Tips

* 時間は管理できない、仕事を管理することでパフォーマンスを上げる
    * 質 x 量 / 投下時間
* 強く影響を与えるものから終わらせると早く仕事が終わる。
* 5分300分 の法則
    * 人間が一生懸命 5分考えて出した output と 300分考えて出した output は同じ。

* ToDo 一覧化
* 重要・緊急マトリクス
* PDCA 管理
* いかに前回よりもうまくやるか
* 伝える技術


* SMART
    | Letter | Description |
    |:--|:--|
    | S | 具体的 |
    | M | 測定可能・定量的 |
    | A | 納得している、達成可能 |
    | R | 現実的 |
    | T | 期限・タイミング |
